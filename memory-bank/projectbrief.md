# Project Brief

Leveraging the Google Agent Development Kit (ADK) to build robust, enterprise-grade, and production-ready AI agent applications.

## Core Goals
- Demonstrate building AI agents using Google ADK.
- Implement production-ready patterns like ReAct and Progressive Disclosure.
- Integrate with observability tools (OpenTelemetry).
- Provide a blueprint for industrial-grade AI agent architecture.

## Key Agents
- **NumberGuesser Agent**: ReAct (Reasoning + Acting) demonstration.
- **Progressive Disclosure Agent**: Multi-agent system for complex task decomposition.
- **PD Troubleshooting Agent**: Advanced diagnostics using ADK features and Progressive Disclosure.

## Target Audience
Developers looking to build enterprise-grade AI applications with rigorous engineering standards.
